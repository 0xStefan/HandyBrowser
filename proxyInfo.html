Fun fact: you can use the Handshake backend of HandyBrowser in your favorite web browsers on the same local network (meaning your mobile too).
How-to: First, start HandyBrowser to run the Handshake Proxy backend. (You can close HandyBrowser anytime after it boots). Then:
Firefox Instructions Chrome (Windows) Instructions Chrome (Linux) Instructions iOS Instructions Android Instructions
My IP: 127.0.0.1
Port: 5301

Firefox

  • Goto Options
  • Search for "Proxy"
  • Click on the "Settings..." button in the Network Settings results
  • Enter your IP and port from above like so:
  • Now search in options for "Certificates", and click the "View Certificates..." Button
  • Download your Handshake Proxy Certificate here, and then in Firefox, click "Import..." to import the downloaded certificate.
  • When importing, you will check the box for "Trust this CA to Identify Websites" like this:

  • And then click "OK"
  • Now visit a Handshake enabled site, like: http://welcome.nb/, or https://3b/

Chrome (Windows)

  • Goto Settings
  • Search for "proxy"
  • Click the option for "Open your Computer's Proxy Settings
  • Scroll down to "Manual Proxy Setup"
  • Click the toggle for "Use a proxy server"
  • Address: localhost
  • Port: 5301
  • Click "Save"
  • Resolve Handshake Names like: welcome.nb/, or https://3b/
  • Note: For Handshake names, make sure you add the trailing slash (/) after the TLD like 3b/ or welcome.nb/

Chrome (Linux)

  • Run chrome with the --proxy-server flag like: google-chrome --proxy-server="127.0.0.1:5301"
  • Resolve Handshake Names like: welcome.nb/, or https://3b/
  • Note: For Handshake names, make sure you add the trailing slash (/) after the TLD like 3b/ or welcome.nb/

iOS

  • Open "Settings"
  • Wi-Fi Settings
  • Click the "i" icon next to your wifi network like:
  • Scroll to the bottom of this screen, and tap "Configure Proxy"
  • Set to "Manual" (try Automatic too if youre lucky)
  • Set the IP and port to the address listed on this page
  • Tap "Save" in the top-right
  • Visit a Handshake enabled site like http://welcome.nb./ (notice the "." after the TLD)

Android

  • Open "Settings"
  • Network & internet
  • Wi-Fi
  • Click the settings cog next to your network name
  • Click the "Pencil" icon at the top to edit network
  • Under "Proxy" click on "None" to edit --> Select "Manual"
  • Proxy hostname = IP listed on this page
  • Proxy port = port listed on this page
  • SAVE
  • Visit a Handshake enabled site like http://welcome.nb on any Browser app
